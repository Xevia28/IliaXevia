E-Commerce App Usability Study

Comprehensive usability testing to validate design decisions and identify improvement opportunities

Usability Study Overview

Study Purpose

The usability study was conducted to evaluate the effectiveness of our redesigned e-commerce app interface and identify areas for improvement before development.

Timeline

2 weeks (June 5-19, 2023)

Methodology

Moderated Testing Task-Based Think Aloud Remote Sessions

Participants

20 users (10 existing, 10 new)

Research Objectives

1. Navigation Efficiency

Measure time to complete key tasks (product discovery, cart addition, checkout)

2. Error Reduction

Identify and quantify points of confusion in user flows

3. User Satisfaction

Gather qualitative feedback on overall experience and perceived value

Testing Scenarios

  1. Find a specific product using search and filters
  2. Add multiple items to cart and proceed to checkout
  3. Save an item for later purchase
  4. Find and apply a discount code
  5. Track an existing order

Key Observations

Search Behavior

70% of users tried filters before typing search terms

Checkout Flow

3 users hesitated at payment method selection

Navigation

Saved items was the least discovered feature

Synthesized Insights

Key Findings

Search & Discovery

Users responded positively to visual search but wanted more filter options for specific needs.

Checkout Process

Streamlined checkout was successful but payment options needed clearer presentation.

Saved Items

Wishlist functionality was underutilized due to low visibility in the navigation.

Recommendations

Enhance Search Filters

Add more granular filter options based on user needs identified in testing

Redesign Payment Selection

Make payment options more visually distinct with icons and clearer benefits

Promote Saved Items

Move wishlist to primary navigation and add visual indicators

Simplify Animations

Reduce motion in add-to-cart animation to prevent distraction

Impact Assessment

Expected Improvement

Task success rate: 87% → 92%

Development Effort

2-3 weeks for all recommended changes

Priority

High (affects core conversion funnel)
